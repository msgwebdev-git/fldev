 Приоритетные функции:
  - Переотправка билетов — кнопка для повторной отправки email с билетами
  - Возвраты — оформление возврата с причиной и статусом refunded
  - Фильтр по датам — выбор периода (сегодня, неделя, месяц, диапазон)
  - Поиск по коду билета — найти заказ по конкретному ticket_code

  Удобство работы:
  - История заказов клиента — все заказы с того же email
  - Заметки к заказу — внутренние комментарии для админов
  - История изменений — лог: кто и когда менял статус/email
  - Массовые действия — выбрать несколько заказов → экспорт/изменить статус

  Аналитика:
  - График продаж — по дням/неделям с выручкой
  - Популярные билеты — какие типы билетов покупают чаще
  - Конверсия промокодов — статистика использования

  Продвинутое:
  - Создание заказа вручную — для продаж по телефону/на месте
  - Уведомления — звук/badge при новом заказе
  - Горячие клавиши — быстрая навигация для опытных операторов
  - Печать билета — PDF для физической выдачи
